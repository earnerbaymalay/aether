## [1.0.0-alpha] - 2026-04-07

### Added
- Multi-tier AI routing: TURBO (Llama-3.2-3B), AGENT (Hermes-3-8B), CODE (Qwen-Coder-3B), LOGIC (DeepSeek-R1)
- AetherVault persistent memory vault (Markdown-based, Obsidian-compatible)
- Python agent core with tool-use execution (`aether_agent.py`)
- Toolbox with 10+ tools: battery, date, file listing, Obsidian integration, web search/read
- Skill Marketplace for drop-in AI behavior modules
- Swarm Orchestrator for multi-agent task solving
- Background Sentinel for passive system monitoring
- Debug Console with self-healing diagnostics
- Librarian for knowledge vault auditing
- Hardware benchmark tool (`bench.sh`)
- Guided one-command installer (`install.sh`)
- Terminal-native TUI with real-time system health display
- Cross-platform documentation (README, USAGE, ROADMAP)
- Apple Edition sibling project ([aether-apple](https://github.com/earnerbaymalay/aether-apple))
